#  Dule Café Website



##  Project Overview

The Dule Café Website is a multi-page café interface built using **HTML, CSS, and JavaScript**, featuring sliders, interactive forms, product pages, confirmation screens, and responsive navigation.

---

##  Project Timeline

### **1️ Project Proposal – (Date: 08-August-2025 )**

* Defined the concept: a café website showcasing products, ordering system, and customer communication.
* Identified required pages:

  * Home
  * About
  * Products
  * Order
  * Confirmation
  * Feedback
  * Contact
  * Enquiry
* Planned features such as sliders, social media links, animated UI, and responsive layout.

---

### **2️ HTML Development – (Date: 27-August-2025)**

Completed structure for all major pages:

* `index.html` (homepage with hero slider)
* `about.html`
* `products.html`
* `order.html` (form for ordering drinks/food)
* `confirmation.html` (order success screen)
* `feedback.html`
* `contact.html` (includes Google Map embed placeholder)
* `enquiry.html`

Key achievements:

* Navigation bar created
* Links between pages established
* Basic layout skeletons completed
* Initial image assets added

---

### **3 CSS Styling – (Date: 26-september-2025)**

Designed the visual appearance of the website using **style.css**:

* Complete responsive design for mobile → desktop
* Hero slider visual styling
* Buttons, cards, product grids
* Color theme, fonts, animations
* Social media icon placeholders
* Layout improvements: spacing, alignment, element sizing
* Added hover effects and smooth transitions

---

### **4️ JavaScript Features – (Date: 19-november-2025)**

Implemented dynamic behaviour inside `script.js`:

* Auto-sliding hero carousel
* Mobile navigation toggle
* Image lightbox popup
* Accordion sections
* Order form price calculator
* Animated confirmation functions
* Automatic date & time update
* Feedback star rating
* Page loader and fade-in animations
